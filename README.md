# xftp_xlqd — 轻量级终端双栏 SFTP 客户端

一个极简的 Linux 终端双栏 SFTP 客户端，基于 C + ncurses + libssh2 开发，运行时内存占用仅 **~6 MB**。

## 特色

- **极轻量** — 纯 C 实现，运行时 RSS ~6MB，二进制仅 39KB
- **双栏界面** — 左栏本地文件系统，右栏远程 SFTP 文件系统
- **键盘全操作** — Tab 切换焦点，方向键导航，快捷键完成所有操作
- **系统路径保护** — 删除 `/etc`、`/bin` 等系统关键目录时二次确认
- **历史主机** — 成功连接的主机自动保存，密码持久化，下次免输
- **实时进度** — 传输文件时显示百分比进度，支持 Esc 取消

## 依赖

```bash
# Ubuntu/Debian
sudo apt install libncursesw-dev libssh2-1-dev libgcrypt20-dev
```

## 运行

```bash
./xftp_xlqd
```

### 一键安装到 PATH

```bash
mkdir -p ~/.local/bin
cp xftp_xlqd ~/.local/bin/
# 之后终端直接输入 xftp_xlqd 启动
```

## 使用指南

### 连接

启动后显示历史主机列表：

```
  ┌──────────────────────────────┐
  │         选择主机              │
  │  1. myserver:22:root         │
  │  2. devbox:2222:xl           │
  │  3. + 新建连接               │
  └──────────────────────────────┘
```

选择历史主机直接连接（密码已保存），或选"新建连接"输入主机/用户/密码。

### 键盘快捷键

| 按键 | 功能 |
|------|------|
| `Tab` | 切换左右面板焦点 |
| `↑` `↓` | 上下移动选中项 |
| `PgUp` `PgDn` | 快速翻页 |
| `Home` `End` | 跳转到列表首尾 |
| `Enter` | 进入选中的目录 |
| `Backspace` | 返回上一级目录 |
| `/` | 编辑当前路径（Enter 导航，Esc 取消） |
| `F2` | 切换显示隐藏文件（. 开头） |
| `F6` | 新建空文件 |
| `F7` | 新建目录 |
| `F8` | 删除文件/目录（含系统路径二次确认） |
| `F9` | 传输文件（上传/下载） |
| `Esc` | 传输中取消 |
| `F10` / `q` | 退出 |

### 传输文件

1. 用 Tab 切换到源面板
2. 选中文件（暂不支持目录传输）
3. 按 F9
4. 确认后状态栏显示实时进度：`下载：45%（2.3M/5.1M） Esc=取消`
5. 按 Esc 随时取消，不完整文件会被清理

## 内存占用

| 版本 | 运行时 RSS | 二进制大小 |
|------|-----------|-----------|
| xftp_xlqd_c（本应用） | **~6 MB** | **~39 KB** |
| FileZilla（C++/wxWidgets） | ~25-50 MB | ~10 MB |
| WinSCP（Wine） | ~30-60 MB | ~15 MB |
| LFTP（C++ CLI） | ~3-5 MB | ~1 MB |

## 配置

- 连接配置：`~/.config/xftp_xlqd/config`
- 主机历史：`~/.config/xftp_xlqd/hosts`
